Decentralized Quasi-Newton Methods

发表于 2024-01-02 更新于 2024-01-03

问题

each node has local variable \(x_{i} \in R^{p}\), communication graph \(G = (\nu, \epsilon)\) ; \(n_{i} = {j | (j=i) v ((i, j) \in \epsilon)}\), \(m_{i} = | n_{i} |\).

我们要提出能够局部更新的的迭代法：

\[ x_{i}(t + 1) = x_{i} + \epsilon(t) d_{i}(t) \]

方法基于下面的local structure：

\[ X^{*} = arg\min_{x \in R^{np}} f(x), with \bigtriangledown _{i} f(x) = \bigtriangledown _{i} f(\hat{x}_{n_{i}}) \]

这里很容易的证明Gradient descent在满足上面的条件下可以局部更新。

BFGS

记号\(v(t) = x(t + 1) - x(t)\), \(r(t) = g(t+1) - g(t)\) 我们根据下面的Gaussian differential entropy来获得\(B(t+1)\)

\[ B(t+1) = arg \min_{Z} tr[B(t)^{-1}Z] - log det[B(t)^{-1}Z] - n\] \[ s.t. \qquad Zv(t) = r(t), z \succeq 0 \]

D-BFGS

记号\(\tilde{ v}_{n_{i}} (t) = D_{n_{i}} [x_{n_{i}}(t + 1) - x_{n_{i}}(t)]\), \(\tilde{r}_{n_{i}}(t) = g_{n_{i}}(t+1) - g_{n_{i}}(t) - \gamma\tilde{ v}_{n_{i}}(t)\)

然后如下选择\(B^{i}(t+1)\)

\[ B^{i}(t+1) = arg \min_{Z} tr[B^{i}(t)^{-1}(Z - \gamma I)] - log det[B^{i}(t)^{-1}(Z- \gamma I)] - n\] \[ s.t. \qquad Z\tilde{ v}_{n_{i}} (t) = \tilde{r}_{n_{i}}(t), z \succeq 0 \]

然后 \[ e^{i}_{n_{i}}(t) = -(B^{i}(t)^{-1} + \Gamma D_{n_{i}})g_{n_{i}}(t) \] \[ d_{i}(t) = \sum_{j \in n_{i}} e^{j}_{i}(t) \]

熵

熵是一种不确定性度量。由下面的离散形式下的公式就能知道概率越小的事件\(-p_{i}ln(p_{i})\)越大。

\[ H(p) = - \sum_{i} p_{i} ln(p_{i}) \]

假如有三个事件概率分别位{1, 0, 0}, {1/3, 1/3, 1/3}熵分别达到最大或最小。

Differential Entropy则是上面公式的连续形式： \[ H(x) = - \int_{- \infty}^{\infty } xp(x)\mathrm{d}x \]

Gaussian Distribution的differential entropy的证明可见于 Differential_Entropy_of_Gaussian_Distribution
